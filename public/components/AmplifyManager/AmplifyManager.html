AmplifyManager Component Demo

This is a standalone audio amplification component for video players.

Video Player Container (Demo)

Component Features:

  • 🔊 AMPLIFY button (grey by default, turns red when active)
  • 🎚️ Slider with gradient colors (green → orange → red)
  • 💡 LED indicator (always visible, shows amplification status)
  • 📊 Percentage display showing current amplification level
  • 🎵 Web Audio API integration for real audio processing
  • 🔄 Automatic reset when loading new videos
  • 📱 Responsive design for mobile devices
